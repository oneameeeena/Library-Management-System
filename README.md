# Library-Management-System

📚 Library Management System

This project is a Python console application designed to manage a library’s collection and loan system.

🚀 Features

Manage book categories (Fiction, Non-Fiction, Science, Biography)

Add, remove, and display books

Search books by title or author

Create and manage loans

Add/remove books from loans with due dates

Cancel loans

Display all loans and borrowed books

Calculate total number of borrowed books per loan

🛠️ Technologies Used

Python 3

Object-Oriented Programming (OOP)

Lists & Dictionaries

Date handling with datetime
